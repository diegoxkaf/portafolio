Conóceme

Soy un desarrollador web y especialista en Web Mapping y Sistemas de Información Geográfica (SIG) con experiencia en la creación de aplicaciones web interactivas y mapas dinámicos. Utilizo tecnologías como HTML, CSS y JavaScript para diseñar y desarrollar soluciones innovadoras que mejoran la visualización y el análisis de datos espaciales. Mi objetivo es combinar la funcionalidad y el diseño para ofrecer experiencias de usuario excepcionales.

Desarrollo Web

Tengo experiencia en desarrollo web, utilizando las tecnologías fundamentales de HTML, CSS y JavaScript para crear sitios web dinámicos y atractivos. A lo largo de mi carrera, he trabajado en diversos proyectos que incluyen diseño y maquetación, interactividad, optimización de sitios web, entre otros. Mi enfoque siempre ha sido crear soluciones web que no solo sean visualmente atractivas, sino también funcionales y eficientes.

Web Maping

He desarrollado a lo largo de mi carrera soluciones web mapping utilizando tecnologías avanzadas como Leaflet, OpenLayers y GeoServer. Con Leaflet y OpenLayers he desarrollado mapas interactivos y responsivos, integrando datos geoespaciales para una visualización intuitiva y eficiente. Con Geoserver he configurado y administrado servidores de mapas para la publicación y gestión de datos geoespaciales, facilitando el acceso y la interoperabilidad de la información
